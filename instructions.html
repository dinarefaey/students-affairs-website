Instructions for users

How to Register :

(1) Go to the navigation bar at the top of the page

(2) Select Register option.

(3) Registeration page will open.

(4)Fill in the required informations field with your information to create an account.

(5)Submit the information that you entered and you'll account will be done.

How to add a student:

(1) Go to the navigation bar at the top of the page

(2) Select Services.

(3)A drop down menu will appear , Select add student option from it.

(4) Add student page will open.

(5) Fill the required information field for the new student to be added such as his number , ID , Name .. etc.

(6) Press save so that all the student's information will be saved.

How to edit your profile :

(1) Go to the navigation bar at the top of the page.

(2) Select profile .

(3) you profile page will open Select the part you want to edit and make changes on it.(You must've already an account and logged in).
